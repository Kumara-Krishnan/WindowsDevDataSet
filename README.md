Project Structure:  
your_project_directory/  
├── sources/  
│   ├── data1.json  
│   ├── data2.json  
│   └── ... (additional JSON files)  
├── docs.db (this will be created if it doesn't exist)  
├── scrape_docs.py  
└── requirements.txt  


Install Dependencies: 
pip install -r requirements.txt


Run the Script: 
python scrape_docs.py
